
# DIRISA Datathon 2025 — Protest Risk Analytics
**Predicting Risk, Protecting Communities**

## Overview
A machine learning project developed during the **DIRISA Student Datathon 2025**, aimed at predicting high-risk protest areas across South Africa using integrated national datasets. The solution supports data-driven risk assessment and decision-making for communities and policymakers.

## Features
- Predictive models identifying high-risk protest areas
- Data preprocessing and feature engineering on socio-economic and geographic datasets
- Model evaluation and comparison across multiple algorithms
- Data visualisations and actionable insights for risk assessment

## Technologies Used
- Python
- Scikit-learn
- Logistic Regression
- XGBoost

## Results
- Achieved **89% prediction accuracy** using XGBoost
